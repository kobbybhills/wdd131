About This Course

Welcome to WDD 131 – Dynamic Web Fundamentals, a course designed to introduce you to the essential skills of modern web development. Throughout this course, you will explore the foundational technologies of the web—HTML for structuring content, CSS for styling and layout, and JavaScript for adding interactivity.

This course covers the fundamentals of dynamic web development, focusing on HTML, CSS, and JavaScript. Students will learn to create interactive web pages and apply dynamic elements to enhance user experience.

My Learning Journey

My journey into web development has been an exciting and transformative experience. Starting with little to no knowledge of coding, I have gradually built my skills through dedication, hands-on practice, and continuous learning. Each project and challenge has pushed me to think critically, solve problems creatively, and develop a deeper appreciation for how websites function.
