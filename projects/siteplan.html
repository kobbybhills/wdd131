Site Name

Name: Learn Asante Twi

This name reflects the purpose of the site, which is to help users learn the Asante Twi language in a simple and interactive way. The name is clear, direct, and easy to remember.

Optional Domain: learnasantetwi.org (not required for this project)

Site Purpose

The website aims to provide simple lessons, vocabulary, pronunciation guides, and interactive practice tools for learners of the Asante Twi language. The target audience includes beginners and non-native speakers interested in learning to communicate in Twi, particularly for travel, cultural exchange, or education purposes.

Scenarios

  • How do I say common greetings like "hello" and "thank you" in Asante Twi?
  • Is there an easy way to practice and test my understanding of Twi vocabulary?

Color Schema

  • Dark Green (#013220): Used for the header, footer, and navigation bar to represent growth and culture.
  • Light Beige (#f5f5dc): Used as the background color for the main content area for readability.
  • Accent Yellow (#f4c542): Used for buttons and highlights to draw attention.

Typography

  • Heading Font: "Merriweather", serif – used for all headings to give a traditional, formal feel.
  • Body Font: "Open Sans", sans-serif – used for paragraph text for readability and modern appearance.
